About this mod  

RSS Input is a mod for phpbb3, which import RSS news feed into your forum.  

Since phpbb3 doesn't provide true cron service, OS scheduler(*nix: cron) support is required for automatic feed import. This mod functions without cron support, but not practical for daily use.

Prerequisite

Experienced phpbb site admin with LAMP server (other OSes: web/database servers and php) knowledge for initial setup.

Check list:
  • database server: MySQL (versions as required for phpBB3 installation)
  • web server: (software and versions as required for phpBB3 installation)
  • php: higher than 5.0 with settings of 'safe_mode = off'. Settings of 'allow_url_fopen = on' OR with working cURL library.
  • OS scheduler: with ability to create/access/execute schedule jobs. For *nix, able to execute `crontab -e` either by console or by wrapper tools.
DO NOT install this mod if you are not sure!
